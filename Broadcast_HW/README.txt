"  Используя Async реализовать многопоточность чтения и отправки данных на сервере
 
в клиенте расширить функционал до возможности переписки с сервером "
--------------------------------------------------------------------------
Многопоточность сделана, посмотрел информацию в интернете про (lock)
Использовал в коде, для избежания одновременной передачи данных двумя или больше клиентами.

Переписка реализована в таком виде (клиент отправил - сервер получил) и одновременно с одного клиента на все остальные, но без клиента отправителя. (если такое не нужно можно просто запустить один клиент)

Общая структура проекта включает: два клиента (одинаковый код, есть возможность добавить ещё клиентов) и один сервер.
